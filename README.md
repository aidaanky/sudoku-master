**Sudoku Master** — полноценная платформа для тренировки мозга с умным тренером, ежедневными челленджами и системой мотивации.

### Для кого это?
- Для тех, кто хочет играть в судоку красиво
- Для новичков, используйте AI Coach объяснит стратегии решения
- Для опытных игроков, включайте уровень Эксперт и Daily Challenge

---

## Функционал

### Игра
- Генератор уникальных судоку на основе алгоритма **backtracking**
- 4 уровня сложности: Лёгкий, Средний, Сложный, Эксперт
- Валидация правил в реальном времени (подсветка ошибок в красном цвете)
- Режим заметок записывай кандидатов в ячейки
- Таймер с паузой
- Кнопка отмены последнего хода
- 3 подсказки на игру

### Встроенный AI Coach
- Анализирует выбранную ячейку и объясняет стратегию
- Определяет технику: Naked Single, Hidden Single, Naked Pair, Pointing Pairs
- Показывает допустимые числа и объясняет почему
- Работает полностью офлайн

### Daily Challenge
- Ежедневное судоку одинаковое для всех пользователей
- Обратный отсчёт до следующего задания
- Таблица рекордов дня
- Streak - подсчет на сколько дней подряд играешь

### Геймификация
- Система достижений, 7 уникальных значков
- Локальная статистика на устройстве: сыграно, победы, процент побед, лучшее время
- Топ-10 ваших игр (для общего лидерборда нужна база данных)

### Монетизация
- Кнопка **Upgrade to Pro ⭐** с описанием возможностей
- Цена: 990₸ в месяц или 6 990₸ в год

### Дизайн
- Тёмная и светлая тема
- Полностью адаптивный (мобильный + десктоп)
- Плавные анимации и конфетти при победе

## 🛠️ Технологии

| Технология | Использование |
|------------|---------------|
| HTML5 | Структура приложения |
| CSS3 + Tailwind CSS | Стили и адаптивность |
| Vanilla JavaScript | Вся игровая логика |

## 📁 Структура проекта

```
index.html
├── <head>        — подключение стилей и шрифтов
├── <style>       — CSS переменные, темы, анимации
├── <header>      — шапка с логотипом и кнопками
├── <aside>       — боковая панель (сложность, статистика, Daily)
├── <main>        — игровое поле и управление
├── модальные окна — победа, Pro, настройки
└── <script>      — вся игровая логика (JS)
    ├── Генератор судоку (backtracking)
    ├── Валидация правил
    ├── AI Coach
    ├── Daily Challenge
    ├── Система достижений
    └── Статистика и localStorage
```
